4个项目 组成 简历数据报告平台。

1、data_create
一个spark项目，用于生成 项目所需的原始数据
2、data_platform
一个web项目，用于统一管理 访问数据报告所需数据的api接口
3、to_b
一个web项目，面向B端用户的 可视化展示的项目
3、to_pm
一个web项目，面向PM端的后台管理项目
